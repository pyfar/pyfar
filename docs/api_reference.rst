API Reference
=============

This section documents pyfar objects and modules. Pyfar contains classes and
functions for the acquisition, inspection, and processing of audio signals and
meta data.

For examples on how to use pyfar refer to the notebooks in the
`examples gallery`_ and the documentation on classes and modules below.
